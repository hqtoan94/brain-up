# Templates

Copy these when starting a new note of each type. The agent uses them too.

- `project.md`
- `area.md`
- `resource.md` — covers both atomic notes (default) and distilled topic pages, via `maturity:`
- `journal-daily.md`
- `person.md`
- `inbox-capture.md`
- `decision.md` — ADR-shape decision record; lives in `brain/resources/decisions/`

These are starting points, not straightjackets. Delete sections you don't need.

# Inbox

Raw captures. Zero-friction dumping ground.

When an idea, quote, task, link, or random thought shows up, it lands here first. No organizing, no categorizing, no pretty formatting. That happens later during **process-inbox** or **weekly-review**.

## Convention

- Filename: `YYYY-MM-DD-kebab-title.md` (the agent's `capture` skill does this for you).
- Minimal frontmatter is fine: `created: YYYY-MM-DD` is enough.
- Keep entries short. If you have a lot to say, dump now — refine later.

## Processing

Run `process my inbox` weekly (or when it exceeds ~15 files). The agent will walk items one-by-one and suggest where each belongs: Project? Area? Resource? Knowledge note? Delete?

**Rule of thumb:** an item should not live here more than two weeks.

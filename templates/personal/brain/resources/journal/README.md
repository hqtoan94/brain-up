# Journal

One Markdown file per day: `YYYY-MM-DD.md`.

Use it for:

- Morning intentions
- Evening reflections
- Quick notes that are date-bound (not timeless ideas → those go to `resources/` as atomic notes)
- Gratitude, lessons, observations

The agent's **daily-journal** skill can open today's file, prompt for reflections, and add structure as you answer.

## Template

See [`../../templates/journal-daily.md`](../../templates/journal-daily.md).

## Linking out

If a journal entry contains a durable insight, **lift it out** into `brain/resources/` as an atomic note (`maturity: atomic`) with a proper title and link back. Journals should be ephemeral; insights should be permanent.

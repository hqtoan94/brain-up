# People

One file per person you want to remember. Friends, colleagues, mentors, leads.

## What belongs here

- Basic facts: how you met, where they work, what they care about
- Conversation log: brief notes after meaningful interactions
- Ongoing threads: things to follow up on
- Links to shared projects or collaborations

## Filename

`firstname-lastname.md`, lowercased, kebab-case. Use a nickname if that's how you think of them.

## Template

See [`../templates/person.md`](../templates/person.md).

## A warning

This folder holds private information about real people. Keep entries factual, kind, and the kind of thing you'd be comfortable with them reading. If genuinely private, prefix with `private-` or keep in a `private/` subfolder (both are gitignored by default).
